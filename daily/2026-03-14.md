# Day 36: Linux CLI Familiarity (free, df)

Today's update focuses on adding more Linux-style aliases and mock commands to make the shell feel more familiar to modern users, even in this 16-bit real-mode environment.

- Added `free` as an alias for the `mem` command (displays conventional memory).
- Added `df` command (mock) to display simulated disk usage for the floppy drive.
- Updated version to v0.1.12.
- Updated `help` documentation.
