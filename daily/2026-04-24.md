## Technical Details
- `irq` command reads the IMR (Interrupt Mask Register) from both the Master PIC (port 0x21) and Slave PIC (port 0xA1).
- Displays the masks as a 16-bit hex value (Slave:Master).
- This is a prerequisite for correctly enabling IRQ 8 (RTC) in upcoming tasks.
