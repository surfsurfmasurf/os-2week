# Day 35: Linux-style Process Management Mocks

Introduced `ps` and `kill` commands to the shell. While the OS currently only supports a single-tasking environment, these mocks lay the groundwork for a future task manager and provide a more familiar "Linux-style" CLI experience.

- Updated version to v0.1.11.
- Added `ps` command to display a static process list.
- Added `kill` command (mock) to simulate process termination.
- Updated `help` documentation with new commands.
