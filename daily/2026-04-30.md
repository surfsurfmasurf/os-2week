# Day 85: Basic Command Logic

## Changes
- Implemented primitive command handling in `kernel_main`.
- Added 'c' key shortcut to clear the screen and reset the prompt.
- Bumped version to `v0.0.4`.

## Notes
The keyboard input is still polled directly from port 0x60 without interrupts (IRQs), so it may miss keys if the CPU is busy, but it works for basic interaction.
