# 2026-04-26 - Day 64: PIC IRQ Masking

Today I implemented the `mask` command in `stage2.asm` to complement yesterday's `unmask` command. This allows full manual control over the PIC (Programmable Interrupt Controller) interrupt masks from the shell.

## Changes
- Added `mask <irq_hex>` command to the shell.
- Implemented logic to set the mask bit (disable IRQ) for Master/Slave PICs based on the IRQ number (0-15).
- Updated the `help` command message to include both `mask` and `unmask`.
- Added success and help messages for the `mask` command.
