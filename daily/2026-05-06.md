## Progress
- Implemented 'reboot' command in the kernel.
- The 'reboot' command (shortcut 'r') sends 0xFE to the Keyboard Controller (port 0x64), which pulses the CPU reset line on most x86 systems.
- Updated kernel version to v0.0.8.

## Technical Details
- Keyboard Controller (8042) command 0xFE is "Pulse Output Port".
- The output port's bit 0 is typically connected to the CPU reset line.
