# Day 34: MS-DOS Compatibility Polish

Added `type` as an alias for `cat` to align closer with MS-DOS command expectations.
The `cat` command (previously added for Linux-style usage) now has its DOS counterpart, allowing users to display sector contents using either command.

- Incremented version to v0.1.10.
- Updated `help` message to include `type`.
- Added `cmd_type` logic.
